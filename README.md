Hra inšpirovaná od originálnej hry Centipede.

# Ovládanie
- menu: ESCAPE

### Vesmírna loď:
- Posun hore: šípka hore
- Posun dole: šípka dole
- Posun vpravo: šípka vpravo
- Posun vľavo: šípka vľavo
- Streľba: SPACE
